Descripción Técnica

AYNI-BOT es un sistema robótico flotante diseñado para el monitoreo continuo de la calidad del agua en piscinas, lagunas, reservorios y estanques. Integra sensores fisicoquímicos, procesamiento embebido y transmisión IoT.

Arquitectura del Sistema

Módulo de Sensado
  • Sensor de pH
  • Sensor de Cloro
  • Sensor TDS
  • Sensor de Turbidez
  • Sensor de Temperatura
Módulo de Procesamiento
  • Filtrado de señal
  • Conversión ADC
  • Calibración de sensores
  • Validación de datos
Módulo de Comunicación
  • Transmisión IoT en tiempo real
  • Envío estructurado de datos
  • Visualización remota
Sistema Energético
  • Panel solar
  • Batería de litio recargable
  • Regulador de carga
  • Optimización de consumo energético

Funcionamiento

  1. El robot flota de forma estable.
  2. Los sensores capturan variables periódicamente.
  3. El microcontrolador procesa las señales.
  4. Se transmiten los datos a la plataforma remota.

Aplicaciones

  • Monitoreo ambiental
  • Control de piscinas
  • Gestión hídrica municipal
  • Proyectos educativos STEAM

Beneficios de AYNI-BOT

  • Monitoreo 24/7
  • Datos en tiempo real
  • Reducción de inspección manual
  • Mayor precisión en mediciones
  • Sistema energéticamente autónomo
  • Escalable para futuras mejoras
